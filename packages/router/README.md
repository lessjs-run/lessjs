# @openelement/router

Route support for the openElement framework.

This package provides:

- File-system route definition and pattern translation.
- Locale-aware path normalization.
- Client-side router helpers.
- Page loader utilities.

Most users should start from `@openelement/app` or `@openelement/element`;
`@openelement/router` is used behind the framework layer.

## Install

```bash
deno add jsr:@openelement/router
```
