# @openelement/style-sheet

CSS-in-JS style sheet utilities for openElement Web Components.

> v0.39 surface: advanced primitive. First-run examples should import
> `StyleSheet` through `@openelement/runtime`.

`@openelement/style-sheet` provides helpers for managing adopted style sheets
and CSS construction in Shadow DOM contexts, used internally by the openElement
rendering pipeline.

## Install

```bash
deno add jsr:@openelement/style-sheet
```

## Exports

| Path | Description                               |
| ---- | ----------------------------------------- |
| `.`  | Style sheet construction and adoption API |
