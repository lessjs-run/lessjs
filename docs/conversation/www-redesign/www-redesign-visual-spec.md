## 8. 响应式断点规范

| 断点    | 宽度      | 关键调整                            |
| ------- | --------- | ----------------------------------- |
| Desktop | > 960px   | 完整布局，侧边导航，双栏代码        |
| Tablet  | 601-960px | 侧边导航折叠为汉堡菜单，单栏代码    |
| Mobile  | ≤ 600px   | 全宽布局，Hero 字号缩小，Bento 单列 |

### 8.1 移动端特殊处理

- 首页多框架区域: 交互组件和代码纵向堆叠
- Guide 页面: 侧边导航折叠为抽屉式
- Blog TOC: 隐藏或折叠为顶部下拉
- 卡片网格: 单列布局
- 代码块: 水平滚动，字号缩小
