# v0.25.0: Declarative DX

> **Released**: 2026-05-29 | **Previous**: v0.24.4 | **SOPs**: 14

## Summary

v0.25.0 delivers 14 SOPs of declarative DX improvements, internal cleanup,
and SSG robustness fixes. `lessPipeline()` becomes the sole build entry,
`less()` is removed, route type generation is introduced, and SSG DSD
output is restored after a critical tagName detection regression.

## Declarative Build Pipeline (SOP-001)

- `lessPipeline(config: PipelineConfig): Plugin[]` — single public build entry
- Config shape: `{ routes, i18n, output, island, pwa, viewTransition, headExtras }`
- Maps to `FrameworkOptions` internally — all existing build code reused
- `less()` removed from public API — zero backward compat

## Route Type Generation (SOP-002)

- `.less/routes.d.ts` auto-generated from `[param]` patterns
- Build-time type safety for route parameters

## Type Declarations (SOP-003/004)

- `static head?: HeadConfig` on DsdElement
- `static client?: ClientConfig` on DsdElement
- Island strategy declarations moved from runtime to build-time

## SignalContext (SOP-005)

- `createContext()` / `provideContext()` / `consumeContext()` — ~30 lines, zero deps
- DOM-tree-based context propagation for theme/locale sharing

## Cleanup & Hardening

- **CSS token convergence** (SOP-006): 20→2 `openPropsTokenSheet` imports
- **`as any` hardening** (SOP-008): 21→0 in `core/src`
- **test-utils.ts** (SOP-009): Single re-export point
- **`less()` @deprecated removal** (SOP-010): Fully removed, not just deprecated
- **island.test.ts cleanup** (SOP-011): 0 old API names
- **`_dsdHydrated` removal** (SOP-012): Unified `_renderOrHydrate()` path
- **String→JSX migration** (SOP-013): 14 pages converted
- **Full regression** (SOP-014): 943 tests pass, all gates green

## SSG Robustness

- **tagName detection**: Reverted from `import()` to regex source scan — `import()` failed for routes with Vite virtual module dependencies
- **SSG smoke test**: Always rebuild (no stale cache skip)
- **DSD output**: Restored from 3→5 DSD markers in generated pages
- **customElements guard**: 4 island components now idempotent

## Breaking Changes

```diff
- import { less } from '@lessjs/adapter-vite';
+ import { lessPipeline } from '@lessjs/adapter-vite';

- less({ routesDir: 'app/routes', island: { upgradeStrategy: 'idle' } })
+ lessPipeline({ routes: { dir: 'app/routes' }, island: { dir: 'app/islands', upgradeStrategy: 'idle' } })
```
