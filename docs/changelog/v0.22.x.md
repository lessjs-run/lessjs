# v0.22.x Changelog

> Architecture Integrity — [ADR-0049](../adr/ADR-0049-architecture-debt-first-roadmap-reset.md)

## v0.22.0

### Changed

- **Consumer Surface Cleanup** ([SOP-001](../sop/v0.22.0/SOP-001-consumer-surface-cleanup.md)): Generated projects now import only `@lessjs/app` + `@lessjs/core` + `@lessjs/ui`. Framework transitives (parse5, entities, hono) resolved by ADR-0047 pre-resolution.
- **Package Boundary Repair** ([SOP-002](../sop/v0.22.0/SOP-002-package-boundary-repair.md)): `build-types.ts` and `virtual-ids.ts` moved from `@lessjs/core` to `@lessjs/adapter-vite`. StyleSheet de-duplicated (canonical: `@lessjs/style-sheet`, core re-exports). UI escape helpers extracted to shared module.
- **Adapter Decomposition** ([SOP-003](../sop/v0.22.0/SOP-003-adapter-vite-decomposition.md)): `@lessjs/adapter-vite` index.ts split from 841 lines into 5 modules: `less-plugin.ts`, `head-injection.ts`, `subpath-resolver.ts`, `optional-package-stubs.ts`, `phase-context.ts`.
- **Signals Facade** ([SOP-004](../sop/v0.22.0/SOP-004-signals-schema-validation-hardening.md)): `SignalEngine` interface defined. Experimental alien-signals engine behind `LESSJS_SIGNALS_ENGINE=alien` flag. No public API changes.
- **Schema Unification**: Hub `CompatibilityTier` now imports from `@lessjs/core` (single source of truth).
- **Hub Scanner Fix**: `records.push(null!)` replaced with typed pre-allocation.

### Removed

- `@lessjs/core` no longer exports build contracts (`build-types`, `virtual-ids`)
- `@lessjs/ui` no longer has duplicate `_esc`/`_escAttr` implementations per component
- `@lessjs/style-sheet` no longer has duplicated implementation in core

### Upgrade Notes

- Consumer `deno.json` imports reduced from 18→3. Existing projects: remove `parse5`, `entities`, `hono`, `@lessjs/signals/framework`, `@lessjs/core/navigation`, `@lessjs/ui/` from imports.
- `build-types` and `virtual-ids` imports should use `@lessjs/adapter-vite/build-types` and `@lessjs/adapter-vite/virtual-ids`.
- Signals package continues to work as before. Alien engine is experimental only.

## Breaking Changes

None (public API unchanged, dependency references adjusted within workspace).
