# Changelog - v0.29.2

Release date: 2026-06-03

## Summary

v0.29.2 is a hygiene patch after the v0.29.1 simplification. It removes orphan
import map entries, updates test references to deleted files, fixes stale
comments, and updates the public API documentation.

## Changes

- **Root import map**: removed orphan `@openelement/core/api` entry pointing to
  deleted `api.ts`.
- **SSG package resolver**: removed `api: 'src/api.ts'` mapping to deleted file.
- **Create CLI test**: updated references from deleted `render-instantiate.ts`
  to `render-ir.ts`.
- **SSG package resolver test**: updated virtual module load from deleted
  `jsx-render-string.ts` to `render-ir.ts`.
- **Stale comments**: updated `security.ts` header to reference `render-dsd.ts`
  instead of removed `render-instantiate.ts`.
- **API docs**: replaced `renderToString` with `renderDsdTree` in English and
  Chinese API guides (`www/content/guide/en/api.md`, `www/content/guide/zh/api.md`).

## Breaking Changes

None.
