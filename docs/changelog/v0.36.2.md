# v0.36.2 Changelog

## Summary

v0.36.2 completes the SSG bridge migration: `@openelement/ssg` now owns the
Vite-free SSG render and postprocess core while `adapter-vite` remains the Vite
orchestration shell.

## Changed

- Moved the Vite-free `ssgRender()` pipeline into `@openelement/ssg`.
- Moved SSG page output, bundle, render options, and evidence types into
  `@openelement/ssg`.
- Moved HTML postprocess helpers for client scripts, CSP, DSD polyfill, View
  Transitions, and Speculation Rules into `@openelement/ssg`.
- Kept adapter-vite compatibility re-exports for the migrated SSG APIs.
- Updated adapter-vite build orchestration to delegate SSG render execution to
  `@openelement/ssg`.
- Unified ISR manifest entries around `createIsrCacheKey()`.
- Added package-local README/license and dependency metadata for
  `@openelement/ssg`.

## Tests

- Added direct `packages/ssg` render and postprocess tests.
- Added adapter bridge compatibility tests.
- Added FileIsrCache persistence coverage.
- Added Streaming DSD Web `Response` consumption coverage.
- Recorded v0.36.2 AutoFlow evidence with one merged low-risk cell.

## Deferred

- Complete adapter-vite SSG file ownership migration remains v0.37.0.
- Server/data/UI/starter/Hub product closure remains v0.37.0.
- Firefox/WebKit expansion remains a v0.37 blocker unless the full e2e gate
  proves it during release verification.
