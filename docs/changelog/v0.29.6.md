# Changelog - v0.29.6

Release date: 2026-06-04

## Summary

v0.29.6 closes architecture debt across the v0.29.x cleanup arc. Eliminates 17 type escapes, deletes 40+ lines of duplicate code, centralizes hub constants, and winds down the `render(): string` path.

## Changes

### Type escape elimination

- **island.ts**: Constructor metadata stamping replaced with WeakMap via `getIslandMeta()`.
- **signal-context.ts**: DOM symbol-keyed signal storage replaced with WeakMap. Uses `WritableSignal<T>`.
- **render-dsd.ts**: `DsdComponentConstructor` type replaces 3 inline casts.
- **vnode.ts**: `RenderFn` type added to `VNode.children`, eliminating 4 render function casts.
- **compat-check**: Production `as any` replaced with precise structural type.
- **adapter-lit, adapter-vanilla**: Styles property cast simplified.
- **adapter-vite**: Plugin cast simplified.
- **@lessjs/app**: Removed unnecessary `LessBuildContext as LessBuildContextClass` alias.
- **cem-parser**: `CompatibilityClassification` imported from `@lessjs/core` instead of local duplicate.

### render() → VNode | null

- `DsdElement.render()` returns `VNode | null` (was `string | VNode`).
- `_renderIntoShadowRoot` simplified from 3 branches to 1.
- `_renderErrorFallback` simplified from 2 branches to 1.
- `onRenderError()` returns `VNode | null`.
- `renderDsd()` retains `typeof result === 'string'` compat path with deprecation warning.

### Code deduplication

- `escapeAttr` / `htmlEscape` duplicated in hub and adapter-vite → imported from `@lessjs/core` (-34 lines).
- `StyleSheetLike` unused import removed from render-dsd.ts.

### Hub centralization

- Created `packages/hub/src/constants.ts` with centralized `VALIDATOR_VERSION`, `LESSJS_UI_VERSION`, `SHOELACE_VERSION`.
- Updated `scanner.ts` to use constants.

### Other

- `ErrorBoundary.render()` updated to `VNode | null`.
- `signal-context.ts` uses `WritableSignal<T>` for mutable signal values.
- `callComponent()` children type includes `RenderFn`.

## Breaking Changes

- `render()` returns `VNode | null`. String returns receive a deprecation warning. SSR path handles strings.

## Verification

- 1297 tests passed, 0 failed.
- `deno task fmt:check` passed (765 files).
- `deno task lint` passed (279 files).
- Local build with DSD verification.
