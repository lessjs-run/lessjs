# Changelog - v0.29.6

Release date: 2026-06-04

## Summary

v0.29.6 closes architecture debt in the render pipeline. Eliminates 17 type escapes, replaces symbol-keyed property stamping with WeakMap, adds precise types for component constructors and For render functions, and removes the last production `as any`.

## Changes

### Type escape elimination

- **island.ts**: constructor metadata stamping (`__island`, `__tagName`, `__layer`) replaced with WeakMap via `getIslandMeta()`.
- **signal-context.ts**: DOM symbol-keyed signal storage replaced with WeakMap. Signal types aligned with `@lessjs/signals`.
- **render-dsd.ts**: `DsdComponentConstructor` type replaces 3 `as unknown as` casts for static `styles` and `tagName` property access.
- **vnode.ts**: `RenderFn` type added to `VNode.children`, eliminating 4 `as unknown as ((item, idx) => unknown)` casts across render-ir, jsx-render-dom, and event-hydration.
- **compat-check**: last production `as any` changed to precise structural type.
- **adapter-lit, adapter-vanilla**: `as unknown as Record<string, unknown>` replaced with `{ styles?: unknown }`.

### Type system improvements

- `ComponentFn` and `ComponentCtor` types defined for VNode.tag.
- `DsdComponentConstructor` type added to `@lessjs/core/types.ts`.
- `RenderFn` type added to `@lessjs/core/vnode.ts`.
