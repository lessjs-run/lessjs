| 项目         | 详情                                                                                                                                                                                                |
| ------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **错误**     | `Rolldown failed to resolve import "@openelement/core" from "virtual:less-ssg-entry"`                                                                                                               |

| **验证**     | 修复后正确生成 35 个别名（含 `@openelement/core`）                                                                                                                                                  |
| **状态**     | ✅ 已修复                                                                                                                                                                                           |
