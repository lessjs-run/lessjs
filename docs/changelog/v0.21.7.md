# v0.21.7 - 2026-05-24

## Summary

v0.21.7 is a JSR consumer-build hotfix for the SSG resolver path. v0.21.6 fixed
the create CLI metadata lookup, but generated projects could still fail during
`deno task build` because `@lessjs/adapter-vite` resolved `@lessjs/core`
subpaths to invalid JSR URLs and did not resolve relative imports inside the
virtual core module.

## Changes

### Adapter Vite SSG Core Resolver

- `less:ssg-core-resolve` now appends `.ts` for `@lessjs/core/*` subpaths when
  fetching source files from JSR.
- Relative imports inside the virtual `\0lessjs:ssg-core/*` module are resolved
  back into virtual core module IDs.
- The JSR core version now follows the current adapter-vite JSR package version
  parsed from `import.meta.url`, with a `0.21.7` fallback.

### Release Version Alignment

- All LessJS packages are aligned to `0.21.7` because JSR packages are immutable
  and `@lessjs/adapter-vite@0.21.6` was already published before this resolver
  fix.
- Internal `@lessjs/*` package references and `deno.lock` were updated to the
  `0.21.7` release line.
