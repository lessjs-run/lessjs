# v0.24.3 - 2026-05-29

## Summary

v0.24.3 is a consolidation release. It hardens the JSX+Signal infrastructure introduced
in v0.24.2, extracts signal utilities from the legacy template module, adds direct
renderer tests, and aligns all docs to the new component model.

## Changes

- **Signal-like module**: `isSignalLike` and `unwrapSignalLike` extracted from `template.ts`
  to neutral `signal-like.ts`. JSX renderers no longer depend on legacy template module.
- **Renderer hardening**: `applyProps` (renderToDOM) and `serializeAttrs` (renderToString)
  now auto-unwrap Signal values in attributes, style objects, and SVG properties.
  No more `[object Object]` from `<div title={signal('x')}>`.
- **New tests**: 38 direct renderer tests covering text/Signal children, SVG namespace,
  event binding with AbortController cleanup, Fragment, className, boolean attrs.
- **API Surface**: `isSignalLike`, `unwrapSignalLike`, and `SignalLike` type now exported
  from `@lessjs/core` via `signal-like.js`.
- **Docs alignment**: CHANGELOG, release notes, and version references updated to v0.24.3.
