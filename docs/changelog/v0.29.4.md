# Changelog - v0.29.4

Release date: 2026-06-03

## Summary

v0.29.4 closes the last architecture debts before JamFullStack work begins.
`callComponent()` now filters prototype pollution keys. `renderDsd()` accepts
light DOM children at the IR level, eliminating fragile string manipulation
on rendered HTML.

## Changes

- **callComponent prototype pollution**: added `DANGEROUS_KEYS` filter in
  `callComponent()` (`render-ir.ts`), matching the existing guard in
  `injectProps()` (`render-dsd.ts`).
- **renderDsd lightDom at IR level**: `renderDsd()` now accepts an optional
  `lightDom: RenderNode[]` option. Light DOM children flow through
  `wrapDsdOutput` → `dsdHostNode.light` → `serializeRenderNode`.
- **Deleted `mergeDsdHostHtmlWithLightDom`**: removed the fragile HTML string
  manipulation function (`lastIndexOf` on HTML) from `render-ir.ts`.

## Breaking Changes

None. `renderDsd()` signature is backward-compatible — `lightDom` is optional.
