# v0.17.1 — Cross-Runtime adapter-vite

**Release date**: 2026-05-16

## Summary

Eliminates the last 2 Deno-only API calls in adapter-vite's SSG CLI, making
`lessjs build` runnable under Node.js and Bun without a Deno runtime.

## Changes

- **Compat**: `Deno.build.os === 'windows'` → `process.platform === 'win32'` in `adapter-vite/src/cli/ssg.ts` (line 72)
- **Compat**: `Deno.exit(1)` → `process.exit(1)` in `adapter-vite/src/cli/ssg.ts` (line 114)
- **Verify**: `deno task build` passes
- **Verify**: `deno task test:e2e` passes
- **Verify**: 522 tests passed, 0 failed; fmt / lint / typecheck all clean

## Breaking Changes

None. This is a patch release — internal compat only, no API change.

## Known Issues

- `Deno.readTextFileSync` in `workspace-alias.ts` remains but is guarded with
  `typeof Deno !== 'undefined'` check and `node:fs` fallback — no action needed.
