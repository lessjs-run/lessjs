# Changelog - v0.29.3

Release date: 2026-06-03

## Summary

v0.29.3 resolves CodeQL security scanning alerts found after the v0.29.1-0.29.2
simplification and release pipeline. 5 real issues fixed, 11 false positives
dismissed.

## Changes

- **render-dsd.ts**: `hasError` dead assignment fixed on render catch path.
- **render-dsd.ts**: Type comparison order flipped in `describeRenderValue`
  (`typeof null === 'object'` quirk).
- **dsd-element.ts**: Two `innerHTML` assignments in fallback/render paths
  explicitly wrapped with `trustRenderHtml()` to mark trust boundaries
  and resolve CodeQL HTML-injection alerts.
- **registry component**: Deleted unused `pkgRoute` variable.

## Breaking Changes

None.

- `deno task fmt:check` passed.
- `deno task lint` passed.
- `deno task typecheck` passed (19/19 packages).
- `deno task test` passed (1317 tests).
- All 19 packages bumped to `0.29.3`.
