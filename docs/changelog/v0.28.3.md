# Changelog - v0.28.3

Release date: 2026-06-02

## Summary

v0.28.3 closes the v0.28 pre-freeze work: router/runtime/protocols test floors
are added, Hono and Playwright lockfile drift is removed, MDX support is wired
through the existing DSD rendering path, and all 19 packages are bumped to
`0.28.3`.

## Changes

- **Test floor**: Added 104 router tests, 90 runtime tests, and 101 protocols
  validator tests to cover routing patterns, runtime facade behavior, and
  manifest/protocol validation edge cases.
- **Protocols validators**: Added `@lessjs/protocols/validators` with runtime
  validation helpers and package exports.
- **MDX design**: Added `docs/adr/ADR-0072-mdx-in-lessjs.md`, recording the
  decision to compile MDX at build time and feed it into the existing JSX/VNode
  and `renderDsd()` pipeline.
- **Content MDX support**: Added `packages/content/src/mdx/` with
  `compileMdx()` and frontmatter-aware module metadata.
- **Adapter Vite MDX plugin**: Added `lessMdx()` / `mdxPlugin()` exports and
  inserted the MDX plugin into the Less Vite plugin stack.
- **Docs demo**: Added `www/app/routes/guide/mdx.tsx` and
  `www/content/mdx/example.mdx` as the public docs-site demonstration path.
- **MDX verification**: Added package-level MDX compile tests, Vite plugin
  transform tests, and a docs snapshot test for DSD-compatible MDX output.
- **Lockfile closure**: Rebuilt `deno.lock` after deleting stale dependency
  state. Hono resolves to `4.12.23`; Playwright and Playwright Core resolve to
  `1.59.1`.
- **P2 cleanup**: `tsconfig.json` documents its editor-only purpose,
  `lint.exclude` matches `fmt.exclude`, and `.github/agents/README.md`
  documents the retained agent prompt assets.
- **Package bump**: All 19 `packages/*/deno.json` versions are `0.28.3`, and
  internal `jsr:@lessjs/*` ranges are updated to `^0.28.3`.
