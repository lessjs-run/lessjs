# v0.26.1 Changelog

## Breaking Changes

None.

## Features

- **Real DOM Signal Binding (ADR-0058)**: `applyProps` auto-creates `effect()` bindings for signal-valued props. Signal changes update DOM attributes directly without re-running `render()`.
- **Children Dynamic Text (ADR-0059)**: `{sig}` in JSX children auto-creates TextNode + effect binding. Dynamic text updates without re-render.
- **`<Show when={signal}>` (ADR-0059)**: Conditional rendering control flow component. Swaps DOM children when signal changes.
- **`<For each={signal}>` (ADR-0059)**: List rendering control flow component. Reconciles DOM children from a signal-driven array.

## Removed

- `DsdElement._patchChildren()` — VDOM diff method
- `DsdElement._vnodeEffectDispose` field — disposers now managed by AbortController
- `effect(() => render())` callbacks in `_renderIntoShadowRoot` and `_hyrateExistingDom`

## New Public API

- `Show` (Symbol) — `@lessjs/core` / `@lessjs/core/jsx-runtime`
- `For` (Symbol) — `@lessjs/core` / `@lessjs/core/jsx-runtime`
