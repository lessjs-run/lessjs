# v0.27.0 Changelog

## New Package: @lessjs/router

- `@lessjs/router/client-router` — `Router` 类封装全部客户端路由（URLPattern）
- `@lessjs/router/define-routes` — `RouteConfig` 类型 + `defineRoutes()` 工具
- `@lessjs/router/pattern-translate` — `toHono()` + `toURLPattern()` 模式翻译
- `@lessjs/router/page-loader` — MD 内容加载器

## Framework

- **Router class (ADR-0063)**: less-layout 的 `_locale()`, `_locales()`, `_otherLocalePath()`, `_otherLocaleLabel()`, `_localizePath()`, `_updateLangSwitch()`, `_routeParams()` 全部替换为 `this.routing.locale` / `.path` / `.localize()` / `.switchPath()` / `.switchLabel()` / `.updateSwitch()` — 删除 ~150 行手写字符串解析
- **innerHTML prop (ADR-0064)**: JSX 支持 `innerHTML={html}` 作为构建时静态内容注入。SSR: `renderToString` 当 raw children 渲。CSR: `applyProps` 设 `el.innerHTML`。仅限 `@lessjs/content` 管道输出
- **OpenProps tokens**: +8 语义 token（`--success`, `--warning`, `--info`, `--*-subtle`, `--overlay`），含 dark mode 变体。删除 49 个零使用 token（indigo-*, shadow-2/3, 旧 typography 别名等），openPropsTokenSheet 300→110 行
- **globalThis.location SSR-safe**: Router `#parseUrl()` 先检查 `globalThis.location` 是否存在（Deno SSR → 无）
