## Implementation Pattern

Each route file should follow the pattern used in `guide/getting-started.ts`:

```typescript
override render() {
  return this._getLocale('zh') === 'en' ? this._renderEn() : this._renderZh();
}

private _renderZh() { /* Chinese HTML */ }
private _renderEn() { /* English HTML */ }
```

For pages that share a single `render()`, the locale check should be added and the existing content split into the `_renderEn()` method.
