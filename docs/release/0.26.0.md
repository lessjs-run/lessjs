# v0.26.0: Framework Decoupling

> **Released**: 2026-05-29
> **Previous**: v0.25.0

## Summary

v0.26.0 removes Vite virtual module coupling, extracts island transform to
`@openelement/core`, introduces zero-bundler dev server, and fixes SSG rendering.

## Breaking Changes

None. Virtual module imports (`virtual:less-nav`, `virtual:less-blog-data`)
are still supported in SSG bundles. Route files should migrate to new imports:

```diff
- import { headerNav } from 'virtual:less-nav';
+ import { headerNav } from '@openelement/content/nav';
```

## Features

### Data Pipeline Decoupling

- 60 route files migrated from virtual modules to framework ESM exports
- New writer utilities: `@openelement/content/nav-data`, `@openelement/content/blog-data`, `@openelement/i18n/data`
- SSG bundle preserves virtual modules for zero-breaking-change migration

### Island Transform Extraction

- `transformIslandSource()` → `@openelement/core/island-transform` (pure, zero Vite)
- Vite plugin reduced to thin wrapper (69 → 36 lines)

### Zero-Bundler Dev Server

- `deno task dev:fast` — Deno.serve + Hono, ~100ms cold start
- No Vite dependency, no virtual module resolution overhead

### SSG Robustness

- Adapter stubs now provide real class exports (not `undefined`)
- React showcase island guards against externalized adapter imports
