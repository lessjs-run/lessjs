# v0.29.2 - Post-Simplification Hygiene

Release date: 2026-06-03\
Changelog: [v0.29.2](../changelog/v0.29.2.md)\
SOP: [SOP-008](../sop/v0.29.2/SOP-008-post-simplification-hygiene.md)\
Supersedes: [v0.29.1](./v0.29.1.md)

v0.29.1 deleted 7 source files and 3 subpath exports. A second full-byte scan
found orphan import map entries, stale test fixtures, outdated comments, and API
documentation still documenting the removed `renderToString`. v0.29.2 cleans
these up.

## What Changed

- Removed orphan `@lessjs/core/api` entry from root `deno.json`.
- Removed `api: 'src/api.ts'` from SSG package resolver mappings.
- Updated `create` CLI test references from deleted `render-instantiate.ts`
  to `render-ir.ts`.
- Updated `adapter-vite` SSG resolver test from deleted `jsx-render-string.ts`
  to `render-ir.ts`.
- Updated `security.ts` header comments to reference `render-dsd.ts` instead
  of removed `render-instantiate.ts`.
- Replaced `renderToString` with `renderDsdTree` in English and Chinese API
  guides.

## What Did Not Change

- No package bumps (these are doc/ref fixes only).
- No API surface changes.
- No renderer behavior changes.
