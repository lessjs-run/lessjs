# v0.29.3 - CodeQL Alert Resolution

Release date: 2026-06-03\
Changelog: [v0.29.3](../changelog/v0.29.3.md)\
SOP: [SOP-009](../sop/v0.29.3/SOP-009-codeql-alert-resolution.md)\
Supersedes: [v0.29.2](./v0.29.2.md)

v0.29.3 is a hygiene release resolving CodeQL security scanning alerts
discovered after the v0.29.1 simplification.

## What Changed

- Fixed dead `hasError` assignment in `renderDsd()` catch path.
- Flipped type comparison order in `describeRenderValue` to avoid the
  `typeof null === 'object'` CodeQL flag.
- Wrapped two `innerHTML` assignments in `dsd-element.ts` with
  `trustRenderHtml()` — rendering fallback and CSR re-render paths.
- Deleted unused `pkgRoute` variable in registry component page.
- All 19 packages bumped to `0.29.3`.

## What Did Not Change

- No public API changes.
- No renderer behavior changes.
- No new features.
