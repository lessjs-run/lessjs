# v0.36.3 Release Note - Complete SSG File Ownership Migration

v0.36.3 completes the SSG ownership migration started in v0.36.2.

## Highlights

- `@openelement/ssg` now owns route scanning, route type generation, virtual
  entry generation, generated data resolution, SSG package resolution, render,
  and postprocess logic.
- `@openelement/adapter-vite` is reduced to Vite build orchestration glue.
- Bridge-only adapter-vite compatibility exports were removed once new SSG
  import paths were stable.
- All 20 packages were aligned to `0.36.3`.

## Verification Basis

- `deno task graph:check`
- `deno task build`
- `deno task test:e2e`
- v0.36.3 SOP task completion
- v0.36.3 NextVersion acceptance criteria

## Deferred

- Firefox/WebKit browser proof moved to v0.36.4.
- Server/Data/UI product closure remains v0.37.0 scope.
