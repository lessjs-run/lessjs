**关键 API 增强**：

```ts
// 1. 工厂模式：每个测试/子进程可独立 registry
import { createAdapterRegistry } from '@openelement/core/adapter-registry';
const registry = createAdapterRegistry();
registry.register(myAdapter);

// 2. 渲染错误钩子
class MyComponent extends DsdElement {
  protected onRenderError(error: unknown): string | VNode {
    return `<div class="error">Render failed: ${error.message}</div>`;
  }
}

// 3. _bindEvents 选项
this._bindEvents(this.shadowRoot!, { skipSignalHtmlDescendants: true });
```

## Migration Guide

### 1. 升级依赖版本

```json
// deno.json — 所有 @openelement/* 依赖
{
  "imports": {
    "@openelement/core": "jsr:@openelement/core@^0.28.0",
    "@openelement/ui": "jsr:@openelement/ui@^0.28.0"
    // ... 其它包同样 bump
  }
}
```

### 2.（可选）采用 AdapterRegistry 工厂模式

```diff
- // Before: 隐式 module-level 状态
- import { registerAdapter, getAdapter } from '@openelement/core';
- registerAdapter(myAdapter);

+ // After: 显式 factory
+ import { createAdapterRegistry } from '@openelement/core/adapter-registry';
+ const registry = createAdapterRegistry();
+ registry.register(myAdapter);
+ // 同一进程可有多个 registry 实例
```

### 3.（可选）自定义渲染错误降级 UI

```ts
import { DsdElement } from '@openelement/core';

class MyComponent extends DsdElement {
  protected override onRenderError(error: unknown): string {
    if (error instanceof TypeError) {
      return '<div class="fallback">Loading failed</div>';
    }
    return super.onRenderError(error); // 默认: console.error + 空字符串
  }
}
```

### 4. 自定义主题迁移（如有）

```diff
- /* Old: 自定义 LessJS theme tokens */
```

### 5. 自定义 island 改造（如有）

```diff
- // Old: 手动状态 + 手动 DOM
- class Counter extends DsdElement {
-   private _count = 0;
-   connectedCallback() {
-     super.connectedCallback();
-     this.querySelector('button')?.addEventListener('click', () => {
-       this._count++;
-       this.querySelector('.count')!.textContent = String(this._count);
-     });
-   }
- }

+ // New: signal + data-on-* 标记
+ class Counter extends DsdElement {
+   private _count = signal(0);
+   override render() {
+     return (
+       <div>
+         <button data-on-click="increment">+</button>
+         <span class="count">{this._count}</span>
+       </div>
+     );
+   }
+   private increment = () => this._count.value++;
+ }
```
