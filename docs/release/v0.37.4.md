# v0.37.4 Release Note

Hygiene + Pure CSS UI Foundation + Architecture Decoupling

## Highlights

- **CSS UI Foundation**: daisyUI class set as pure CSS via `CSSStyleSheet`,
  DSD shadow DOM compatible, Open Props token powered.
- **Router decoupled from UI**: `@openelement/ui` now 3 deps (was 5).
- **11 deprecated shells removed**: adapter-vite cleanup complete.
- **Core logger unified**: consistent scoped logging across core packages.
- **1600 tests, 0 failures**: +113 new tests across ssg, router, protocols.

## Packages

All 20 `@openelement/*` packages at `0.37.2` (v0.37.x train).

## Migration Notes

- If importing SSG APIs from `@openelement/adapter-vite`, migrate to
  `@openelement/ssg`.
- If using `@openelement/router` via `@openelement/ui`, add a direct
  dependency on `@openelement/router`.
