# v0.29.0 - Structured Renderer IR

Release date: 2026-06-03\
Changelog: [v0.29.0](../changelog/v0.29.0.md)\
Includes: [v0.28.6](./v0.28.6.md)

v0.29.0 turns the renderer cleanup plan into code. LessJS now has an internal
structured renderer model and a shared serializer instead of keeping
`renderToString()`, `renderDsdTree()`, and DSD wrapping as separate string
rendering islands.

## What Changed

- `RenderNode` IR now represents text, element, fragment, trusted HTML, and DSD
  hosts.
- `renderToString()` and `renderDsdTree()` are thin compatibility entrypoints
  over the IR implementation.
- `wrapDsdOutput()` emits a DSD host node through the shared serializer.
- `rawHtml` is explicit trusted HTML, while normal text and `innerHTML` remain
  escaped.
- v0.28.6 release-pipeline cleanup is included as the foundation for this
  renderer line.

## What Did Not Change

- No public renderer API shape changes.
- No UI Shell expansion.
- No package rebrand.
- No bundled sanitizer in `@openelement/core`.
