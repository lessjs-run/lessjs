# v0.26.1 — Real DOM Signal Binding

Release date: 2026-05-30\
ADR: [ADR-0058](../adr/0058-real-dom-signal-binding.md), [ADR-0059](../adr/0059-show-for-control-flow.md)\
SOP: [SOP-001](../sop/v0.26.1/SOP-001-real-dom-signal-binding.md)

## TL;DR

**彻底消灭了 DsdElement 中的 VDOM 循环，实现了完整的 Real DOM Signal 绑定体系。**

Signal 变化不再触发全量 `render()` + DOM 重建。三层覆盖：

| 层                | 实现                                                                            |
| ----------------- | ------------------------------------------------------------------------------- |
| Props             | `applyProps` 自动检测 signal → `effect(() => el.setAttribute(attr, sig.value))` |
| Children 动态文本 | `{sig}` → 自动创建 TextNode + effect 绑定                                       |
| 条件渲染          | `<Show when={sig}><A/><B/></Show>`                                              |
| 列表渲染          | `<For each={sig}>{(item) => <li>{item}</li>}</For>`                             |

## What Changed

### 架构变更

| Before (v0.26.0)                         | After (v0.26.1)                     |
| ---------------------------------------- | ----------------------------------- |
| `effect(() => render() + clear+rebuild)` | signal→DOM 直接属性/文本/子节点绑定 |
| signal 变化 → 全量 render → VNode diff   | signal 变化 → 单属性/单节点更新     |
| render 调用次数 = 1 + N                  | render 调用次数 = 1                 |
| 隐含 VDOM（清空重建 / patch）            | Real DOM 细粒度绑定（对标 SolidJS） |

### 代码变更

| File                                     | Change                                                                                |
| ---------------------------------------- | ------------------------------------------------------------------------------------- |
| `packages/core/src/jsx-render-dom.ts`    | `applyProps` signal 绑定；`renderToDom` children signal 绑定；`Show`/`For` 控制流分支 |
| `packages/core/src/jsx-render-string.ts` | `Show`/`For` SSR 快照展开                                                             |
| `packages/core/src/jsx-runtime.ts`       | 新增 `Show`/`For` Symbol 导出                                                         |
| `packages/core/src/dsd-element.ts`       | 删除 `effect(render)`×2、`_patchChildren`、`_vnodeEffectDispose`                      |
| `packages/core/src/index.ts`             | `Show`/`For` 公开导出                                                                 |

### 新增 API

```ts
import { Show, For } from '@lessjs/core';

// 条件渲染
<Show when={this._loading}><Spinner/><Content/></Show>

// 列表渲染
<For each={this._items}>{(item: Item) => <li>{item.name}</li>}</For>
```
