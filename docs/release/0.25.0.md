# v0.25.0: Declarative DX

> **Released**: 2026-05-29
> **Previous**: v0.24.4

## Summary

v0.25.0 delivers 14 SOPs of declarative DX improvements. `lessPipeline()`
becomes the sole build entry point, route type generation is introduced,
`static head` / `static client` are declared on DsdElement, and SSG DSD
output is restored after a critical tagName regression. `less()` is fully
removed â€” no backward compatibility shipped.

## Breaking Changes

### less() Removal

```diff
- import { less } from '@lessjs/adapter-vite';
+ import { lessPipeline } from '@lessjs/adapter-vite';

- export default defineConfig({ plugins: [less({ routesDir: 'app/routes' })] });
+ export default defineConfig({ plugins: [lessPipeline({ routes: { dir: 'app/routes' } })] });
```

### API Naming (carried from v0.24.4)

```diff
- island(), lessBind(), renderDSD(), renderToDOM(), getSSRProps()
+ defineIsland(), bindEvents(), renderDsd(), renderToDom(), getSsrProps()
```

## Critical Fixes

- **SSG DSD regression**: `import()`-based tagName detection replaced with regex source scan. All 55 page routes had `tagName: undefined` â†’ SSG produced empty DSD output. Fixed.
- **customElements duplicate registration**: 4 island components now guarded with `!customElements.get(tagName)`.
- **SSG smoke test**: No more stale cache â€” test always does fresh build.
