# v0.36.1 Release Note

## Summary

v0.36.1 closes the v0.36 release truth and AutoFlow evidence gap. It is a
stabilization patch, not a v0.37 product expansion.

## Changes

- Fixed AutoFlow generated version-bump tests on Windows by using path-safe APIs.
- Made successful AutoFlow execution end in `merged` instead of stopping at
  `harness:passing`.
- Counted only merged cells as successful evolution work.
- Added ledger metadata required for metrics replay.
- Derived AutoFlow package count from the actual 20-package graph.
- Executed `cell-v0.36.1-001` as a real low-risk AutoFlow cell.
- Recorded `totalCellsAttempted: 1`, `totalCellsMerged: 1`, and
  `firstPassRate: 1` in `docs/autoflow/metrics/v0.36.1.json`.
- Aligned v0.36.0 docs to show delivered work and deferred v0.37 scope.

## Deferred

Full adapter-vite migration, expanded Firefox/WebKit proof, and
server/data/UI/starter/Hub product closure remain in v0.37.0.
