# v0.29.6 - Architecture Debt Closure

Release date: 2026-06-04\

v0.29.6 closes architecture debt in the render pipeline. Eliminates 17 type escapes, replaces symbol-keyed property stamping with WeakMap, adds precise types for component constructors and For render functions.

## TL;DR

- 17 `as unknown as` / `as any` type escapes eliminated
- `__island`/`__tagName`/`__layer` stamping → `getIslandMeta()` WeakMap
- `signal-context.ts` DOM stamping → WeakMap
- `DsdComponentConstructor` + `RenderFn` types defined
- Last production `as any` removed
- 1297 tests, 0 failures

## What Changed

### WeakMap replaces DOM stamping

```ts
// Before: stamping on constructor
(componentClass as unknown as Record<string, unknown>).__island = true;

// After: type-safe WeakMap
_islandMeta.set(componentClass, { isIsland: true, tagName, layer });
export function getIslandMeta(ctor) { return _islandMeta.get(ctor); }
```

### Precise component constructor type

```ts
// Before
const ctor = componentClass as unknown as { styles?: StyleSheetLike | StyleSheetLike[] };

// After
export interface DsdComponentConstructor extends CustomElementConstructor {
  styles?: StyleSheetLike | StyleSheetLike[];
  tagName?: string;
}
const ctor = componentClass as DsdComponentConstructor;
```

### For render function type

```ts
// Before
const renderFn = children[0] as unknown as ((item: unknown, idx: number) => unknown);

// After
export type RenderFn = (item: unknown, idx: number) => unknown;
// VNode.children now includes RenderFn
const renderFn = children[0] as RenderFn;
```
