## Migration Guide

### 1. Update render function call sites

```ts
// Before
import { renderDsdByName } from '@openelement/core';
await renderDsdByName('less-layout');

// After
import { renderDsd } from '@openelement/core';
await renderDsd('less-layout');
```

### 2. Update JSX import source

```ts
// Before
import { jsx, jsxs } from '@openelement/core';

// After
import { jsx, jsxs } from '@openelement/core/jsx-runtime';
```

### 3. Remove parse5 dependency

```json
// deno.json йҲҘ?remove this line
{ "imports": { "parse5": "npm:parse5@7.0.0" } }
```

### 4. Update page component patterns

```tsx
// Before: page wraps itself in <open-layout>
export default class HomePage extends DsdElement {
  render() {
    return (
      <open-layout>
        <h1>Hello LessJS</h1>
        <less-counter client:idle />
      </open-layout>
    );
  }
}
```

```tsx
// After: pages only declare their content
export default class HomePage extends DsdElement {
  render() {
    return <h1>Hello LessJS</h1>;
  }
}
```

The App Shell (`<open-layout>`) is now applied automatically by the Builder.
