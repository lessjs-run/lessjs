# v0.27.0 — Unified Router + Content Architecture + www Overhaul

Release date: 2026-05-30
ADR: [ADR-0063](../adr/ADR-0063-unified-router-hono-urlpattern.md), [ADR-0064](../adr/ADR-0064-static-content-injection.md)
SOP: [SOP-003](../sop/v0.27.0/SOP-003-unified-router.md)

## TL;DR

**新增 `@lessjs/router` 包 — Hono (SSR) + URLPattern (CSR) 统一路由。www 全量翻修：排版、暗色模式、死代码清理、OpenProps 语义化。**

## New Package: @lessjs/router

```
@lessjs/router/client-router    — Router 类：locale/path/localize/switchPath
@lessjs/router/define-routes    — RouteConfig 类型 + defineRoutes()
@lessjs/router/pattern-translate — toHono() + toURLPattern()
@lessjs/router/page-loader      — MD 内容加载器
```

**`Router` 类接管 less-layout 全部路由逻辑：**

- `this.routing.locale` / `this.routing.path` — URLPattern 解析
- `this.routing.localize(path)` — locale 前缀
- `this.routing.switchPath()` / `this.routing.switchLabel()` — i18n 切换
- `this.routing.updateSwitch(root)` — SPA 导航更新

**删除 ~150 行手写字符串解析**（`_locale()`, `_locales()`, `_otherLocalePath()`, `_localizePath()` 等）。

## www Site Overhaul

### Typography System

- `docs/design/typography.md` — 完整排版设计文档
- h1-h6 层级按角色定义（非装饰）
- body 20px → 16px，max-width 720px 提高可读性
- 全部使用 `var(--font-size-N)` / `var(--font-lineheight-N)`

### Dark Mode

- `:root[data-theme="dark"]` 全局 CSS 级联穿透 shadow DOM
- 架构页/roadmap/registry 自定义 `--arch-*` / `--road-*` → 语义 token
- 暗色模式下内容完全可见

### OpenProps Migration

- +8 语义 token：`--success`/`--warning`/`--info` + subtle 变体 + `--overlay`
- -49 死 token（openPropsTokenSheet 300→110 行）
- 全部 `var(--xxx, #fallback)` 清理（0 残留）
- 全部 `--border-futuristic`/`--brand-neon`/`--cyber-green` → 语义 token

### Navigation

- Logo 呼吸灯动画 + `aria-current="page"`
- Header-right 四元素（Search/Theme/Lang/GitHub）统一样式
- `:root[data-theme]` 全局暗色 token 注入

### I18n Fix

- Router `locales` 改为 lazy getter（SSR construct 后 attribute 才设置）
- lang-switch SPA 导航后自动更新 label + href

### Content

- changelog.tsx: 98 行手动 markdown 解析器 → `marked.parse()`
- ADR-0064 批准 `innerHTML={html}` 作为构建时静态内容注入路径
- Blog 内容渲染修复（innerHTML 在 SSR/CSR 双路径一致）

### Dead Code

- 删除 7 个零引用岛屿：demo-*, media-chrome-showcase, react-showcase, counter-island
- 删除 `functions/api/term.ts` + `term-commands.ts`（-444 行）
- Bundle 节省 192 KB（预算警告消失）
