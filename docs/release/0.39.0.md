# openElement v0.39.0

v0.39.0 is the Framework RC + Four-Product Matrix Reset release.

## Highlights

- Establishes ADR-0099: `openElement = Elements + UI + Framework + Protocols`.
- Records Elements as the future `@openelement/elements` / `OpenElement`
  product direction.
- Keeps `DsdElement` out of long-term public compatibility commitments.
- Keeps Web Awesome out of the current UI target.
- Limits heavy-framework island planning to a Vue adapter proof.
- Restores JSR publish as a release exit gate under ADR-0100.

## Framework RC Proof

- `@openelement/create` generated apps now exercise:
  - pages and layouts;
  - islands;
  - API routes;
  - public static assets;
  - SSR/ISR cache intent;
  - Vite build output;
  - Nitro node runtime output.
- `deno task consumer:local` verifies the generated app, Nitro node server,
  `/`, `/freshness`, `/api/health`, and `/openelement-mark.svg`.
