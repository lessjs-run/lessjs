## Summary

v0.24.4 applies the Web Platform naming convention across the entire public
API surface. Function names follow `verbNoun`, acronyms use `PascalCase`
(`Dsd` not `DSD`), and brand prefixes are removed. This is a mechanical
rename with no behavioral changes â€” all 944 tests pass identically.

## Breaking Changes

**No backward compatibility.** Old names removed entirely. No `@deprecated`
aliases shipped. Update imports immediately:

```diff
- import { island, lessBind, renderDSD, renderToDOM, getSSRProps } from '@lessjs/runtime';
+ import { defineIsland, bindEvents, renderDsd, renderToDom, getSsrProps } from '@lessjs/runtime';
```
