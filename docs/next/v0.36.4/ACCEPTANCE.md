# v0.36.4 Acceptance Criteria

## Exit Criteria

1. `test:e2e:browsers` passes on Chromium, Firefox, and WebKit

4. CI workflow includes cross-browser E2E as a gate
5. All 12 release gates pass
6. Packages bumped to 0.36.4

## Evidence Requirements

- E2E test results for all 3 browsers
- Known limitations documented in this NextVersion package
- CI workflow updated
