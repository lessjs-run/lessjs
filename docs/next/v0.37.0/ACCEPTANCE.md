# v0.37.0 Acceptance

v0.37.0 is accepted when repository planning truth is aligned with ADR-0091.

Required evidence:

- ADR-0091 exists and is linked from ADR index, SOP, and NextVersion.
- `docs/conversation/20260609/four-product-roadmap-reset.md` records the
  roadmap decision.
- Roadmap/status/README/website copy no longer present v0.37.0 as a single
  Server/Data/UI closure epic.
- SOP index lists v0.37.0-v0.37.6 with bounded purposes.
- v0.37.0 NextVersion package has all mandatory files.
- Local gates listed in `TEST_MATRIX.md` pass or have documented blockers.

Non-acceptance:

- Any product code implementation in v0.37.0.
- Any claim that light DOM, database adapters, pure CSS UI, protocol ports, or
  full-stack preset work is already implemented by this reset.
