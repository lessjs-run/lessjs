# v0.30.1 SOP

v0.30.1 is the post-freeze cleanup release for openElement.

The governing SOP is:

- [SOP-013: v0.30.1 Clean Architecture Sweep](./SOP-013-islands-fix-openelement-rename.md)

The governing ADR is:

- [ADR-0081: VNode-Only Dynamic UI and Trusted HTML Boundary](../../adr/ADR-0081-vnode-event-unification.md)

prove the active repository has one package identity, one UI naming line, one
dynamic interactive renderer model, and one trusted HTML boundary.
